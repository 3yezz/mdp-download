# EDmdp — Gestionnaire de mots de passe

**Français** | [English](#english)

---

## Français

EDmdp est un gestionnaire de mots de passe sécurisé disponible sur Windows, Linux, macOS et Android.

### Fonctionnalités

- Stockage chiffré des mots de passe
- Génération de mots de passe forts
- Scan de QR codes pour import/export rapide
- Recherche rapide dans les entrées
- Interface réactive adaptée bureau et mobile

### Plateformes

| Plateforme | Format |
|------------|--------|
| Windows | `.exe` (installeur NSIS) |
| Linux | `.tar.gz` |
| macOS | `.dmg` |
| Android | `.apk` |

### Installation

Télécharge la dernière version depuis les [releases GitHub](https://github.com/3yezz/MDP/releases).

### Stack technique

- **Interface** : React 18
- **QR Code** : jsQR
- **Bureau** : Electron
- **Mobile** : Android natif (WebView)
- **Build** : electron-builder, Gradle

---

## English

EDmdp is a secure password manager available on Windows, Linux, macOS and Android.

### Features

- Encrypted password storage
- Strong password generator
- QR code scanning for quick import/export
- Fast search through entries
- Responsive interface for desktop and mobile

### Platforms

| Platform | Format |
|----------|--------|
| Windows | `.exe` (NSIS installer) |
| Linux | `.tar.gz` |
| macOS | `.dmg` |
| Android | `.apk` |

### Installation

Download the latest version from the [GitHub releases](https://github.com/3yezz/MDP/releases).

### Tech stack

- **UI** : React 18
- **QR Code** : jsQR
- **Desktop** : Electron
- **Mobile** : Android native (WebView)
- **Build** : electron-builder, Gradle
